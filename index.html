Alofsto Dashboard
❓ What is it?
The Alofsto dashboard is a fully JS (Javascript) dashboard, with lots of different functions (See the Functions section.) When you click on the bookmark, you will see a menu pop up on the top right of the page. From there, you can see the different functions, and choose which one you want to use.
⬇️ How do I get it?
Simply drag the link below to your bookmarks. If you can't see the bookmark bar, press CTRL+SHIFT+B, or if you are on Mac, press ⌘+SHIFT+B.
Drag this: ➡️ 🚀 Alofsto Dashboard ⬅️
📄 What can it do?
•
📥 Embed - Opens another page in a small window on your current tab
•
🔑 Reveal passwords - Reveals passwords (That look like •••••••) on the current page
•
🏷 Change page title - Changes the title of the current page
•
✏️ Toggle edit mode - Allows you to edit the website, kind of like inspect mode
•
⏱ Page load time - Tells you how long it took to load the current page (Like a wifi speed test)
•
🖱 Autoclicker - The next thing you click on will be clicked very, very fast
